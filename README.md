#### getting started
```
git clone https://github.com/plebbit/ipfs-tracker.git && cd ipfs-tracker
npm install
DEBUG=ipfs-tracker:* npm start
```

#### getting started with docker
```
sudo apt install docker.io
git clone https://github.com/plebbit/ipfs-tracker.git && cd ipfs-tracker
scripts/start-docker.sh
```

#### test
```
npm test
```

or

```
npm run test:watch
```
